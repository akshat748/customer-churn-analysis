# 📊 Customer Churn Analysis

## Overview
Predict customer churn using machine learning.

## Tools
Python, Pandas, Seaborn, Scikit-learn

## Models
- Logistic Regression
- Random Forest

## Key Insights
- Contract type strongly affects churn
- High charges increase churn
